feat(phase-50.3): Analytics dashboard, adaptive autotuning, scheduler extensions + tools page

Phase 50.3 delivers three major agent enhancements:

1. **AB Analytics Dashboard** (Visual CTR insights with Recharts)
   - Event storage: ab_store.py with JSONL persistence
   - Daily CTR aggregation with date filtering
   - API: GET /agent/ab/summary?from=YYYY-MM-DD&to=YYYY-MM-DD
   - Frontend: ABAnalyticsDashboard.tsx with line chart, stats cards, winner display
   - Features: Date range filters, refresh button, responsive design

2. **Adaptive Agentic Feedback Loop** (AI-driven weight optimization)
   - Autotuning service: weights_autotune.py with learning rate alpha
   - Algorithm: new_weight = max(0, base + alpha * hint) then normalize
   - API: POST /agent/autotune?alpha=0.5 (default alpha: 0.5)
   - Frontend: AutotuneButton.tsx with loading state, success/error feedback
   - Safety: Non-negative weights, gradual updates, normalized to 1.0
   - Integration: Dispatches siteagent:layout:updated event

3. **Scheduler Extensions** (YAML policy, manual triggers, audit trail)
   - YAML config: data/schedule.policy.yml (nightly_time, presets, holidays)
   - Enhanced scheduler.py: pick_preset_for_day(date) for day-type selection
   - Manual trigger: POST /agent/run_now?preset=X
   - Audit trail: agent_events.py with JSONL logging
   - API: GET /agent/events?limit=50 for event history
