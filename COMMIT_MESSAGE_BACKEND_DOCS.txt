Implementation Details:
- HMAC token structure: base64url(payload).base64url(signature)
- Payload: JSON with email, role, exp fields
- Signature: HMAC-SHA256(payload, secret)
- Cookie: admin_auth (HttpOnly, Secure, SameSite=None, 7d expiry)
- Domain: .ledger-mind.org (works on all subdomains)

Security Model:
- Stateless: No database, self-contained tokens
- Tamper-proof: HMAC signature validation
- Secure: HttpOnly prevents XSS, Secure requires HTTPS
- Cross-subdomain: SameSite=None + Secure (portfolio + API on different subdomains)
- Time-limited: 7-day expiry with 5-minute clock skew buffer
- Allowlist: Only authorized emails can get admin tokens

Testing Examples:
- Login: POST /api/auth/admin/login?email=you@yourdomain.com
- Check: GET /api/auth/me (with cookie)
- Test: POST /api/layout/autotune (with cookie)

Critical Requirements:
- CORS: allow_credentials=True (required for cookies)
- All backend replicas: Same ADMIN_HMAC_SECRET (critical for multi-pod)
- Cookie domain: .ledger-mind.org (note leading dot for subdomains)
- HTTPS: Required for Secure cookie attribute
